la etiqueta a

es una etiqueta para enlazar recursos para que el usuario pueda navegar de punto a otro punto,es una etiqueta escencial de html 5 y debes saber usarla

en este punto ya uno sabe para que y como funciona,dire alguno atributo que no sabias

  • href:Enlace a la página, documento o recurso que se quiere enlazar. Atributo obligatorio.
  • hreflang:Establece un código ISO 639-1 del idioma del recurso enlazado en el atributo href.
  • download:Establece el nombre del archivo a descargar.
  • target:Establece la ventana en la que se abre el recurso enlazado.
    1. _blank:Abre el recurso en una nueva ventana.
    2. _self:Abre el recurso en la misma ventana.
    3. _parent:Abre el recurso en la ventana padre.
    4. _top:Abre el recurso en la ventana principal.
    5. frame:Abre el recurso en un marco.
  • ping:Notifica a un recurso o enlace concreto que el usuario ha seguido el enlace.
  • rel:Tipo de relación del enlace
    1. alternate:Indica que el recurso enlazado es una alternativa al recurso principal.
    2. author:Indica que el recurso enlazado es el autor del recurso principal.
    3. bookmark:Indica que el recurso enlazado es un marcador de navegación.
    4. help:Indica que el recurso enlazado es una ayuda.
    5. license:Indica que el recurso enlazado es una licencia.
    6. next:Indica que el recurso enlazado es el siguiente recurso.
    7. prev:Indica que el recurso enlazado es el recurso anterior.
    8. search:Indica que el recurso enlazado es una pista de búsqueda.
    9. tag:Indica que el recurso enlazado es una etiqueta.
    10. external:Indica que el recurso enlazado es externo.
    11. nofollow:Indica que el recurso enlazado no debe ser seguido por el navegador.
    12. noopener:Indica que el recurso enlazado debe ser abierto en una ventana nueva.
    13. noreferrer:Indica que el recurso enlazado debe ser abierto sin referencias.
    14. prefetch:Indica que el recurso enlazado debe ser descargado de forma anticipada.
  • type:Pista para el navegador sobre el tipo de recurso enlazado. Atributo no obligatorio.
