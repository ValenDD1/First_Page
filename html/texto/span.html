La etiqueta span se utiliza para grupar otras etiquetas y organizar la información de tu código HTML. Las etiquetas

son una de las etiquetas HTML más populares y se trata de una etiqueta de bloque.

Las etiquetas , al ser elementos en línea, no modifican la disposición de los elementos y mantienen la fidelidad original del párrafo.
