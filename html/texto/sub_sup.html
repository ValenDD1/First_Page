sub y sup

La etiqueta sub se utiliza para poner un texto subscrito

La etiqueta sup se utiliza para poner un texto superescrito

ejemplos:

El resultado de 242 es 576.

Esa información la obtendremos en la variable xa.

Formulas complejas:

El volumen de una esfera es V=4/3πr3

Si buscas un sistema más complejo para representar fórmulas o ecuaciones matemáticas, quizás deberías profundizar en alguno de estos sistemas

Nombre Descripcion
MathJax Motor Javascript para matemáticas. Convierte a SVG o a HTML.
