La etiqueta p

p es una etiqueta de parrafo que entr en la palabra de agrupacion

ejemplo(F12):

Hola me llamo Valentino

aqui un ejemplo de los fragmentos de semantica(linea)

fragmentos de texto
  1. a:sirve para enviar un link
  2. strong:poner a la palabra como importante
  3. em:poner a la palabra con cursiva
  4. b:poner a la palabra con negrita
  5. mark:poner el texto en subrayado con color
  6. i:poner el texto o palabra en italica
  7. u:poner el texto o palabra subrayado
  8. s:poner el texto con una raya
  9. span:poner el texto con una etiqueta
fragmentos con signiicados
  1. sup:sirve para poner un indice para arriba 242
  2. sub:sirve para poner un indice abajo 242
  3. small:sirve para poner un antocacion ,mas pequeno
  4. cite:sirve para poner una cita videojuego: the last of us
  5. q:una cita o frase de otro contenido extraida the last of us
  6. abbr:abreviatura o acronimo js
  7. dfn:termino que posteriormente sera definido css
fragmentos tecnicos
  1. kbd: entrada del informacion del usuario ctrl
  2. samp:salida de informacion de un programa informatico error 5
  3. var:variable nombre
  4. time:hora 12:00
  5. data:informacion equivalente orientado a máquinas. hola
  6. code:fragmento de codigo fuente const y=12;
fragmentos de edicion
  1. ins :Fragmento de texto o información añadida a posteriori de la publicación inicial hola
  2. del :Fragmento de texto o información eliminada a posteriori de la publicación inicial hola
