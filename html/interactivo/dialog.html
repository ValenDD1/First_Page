Dialog

ahora haremos un modal,usando js y css ,hay propiedades con css para poder estilar un dialogo o modal

  • :open : Da estilo a la ventana de diálogo que está abierta
  • :closed : Da estilo a la ventana de diálogo que está oculta
  • ::backdrop : Da estilo al fondo de la página cuando el diálogo está abierto.
