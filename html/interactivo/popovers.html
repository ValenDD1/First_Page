Popovers

Los popover (pop over, salta por encima) son una nueva forma en HTML5 de crear elementos emergentes, es decir, contenido que aparecerá al usuario por encima del contenido que ya estaba visualizando, para así llamar temporalmente su atención y mostrar contenido adicional, de contexto o de ayuda.

Ejemplos:

Alerta has recibido un popver

    para que funcione el popover,usar
  • popovertarget: indica el id del elemento que se va a controlar
  • popovertargetaction: indica la acción que va a realizar, entre show, hide o toggle,por defecto esta en show
  • popover: indica que es un popover

Estilos de css para el popover

podemos mejorar nuestros popovers,incluso hacer animaciones,etc..

    estos son los elementos de css para poder estilar el popover
  • ::backdrop : estila el fondo del popover
  • :popover-open : estila el popover al abrirlo
  • [popover]:estila todo elemento que tenga el atributo "popover"
ojo:tambien el atributo popover tiene como otro parametro el manual hace que en vez de cerrar el popover con un click fuera de su entorno o esc,no se pueda cerrar
