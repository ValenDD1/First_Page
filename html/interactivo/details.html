details

El llamado Acordeón HTML, elemento desplegable o colapsable es un elemento que muestra un texto, forma o icono identificativo que el usuario puede pulsar para desplegar más información, que por defecto está oculta y contraída.

  • details:Etiqueta contenedora de todo el acordeón o elemento desplegable.
  • summary:Título pulsable del acordeón. Aparece siempre, se encuentre desplegado o no.

acordeon exclusivo,name

Es posible crear un acordeón exclusivo con HTML. Este es el nombre que se le da a una serie de acordeones HTML donde sólo se permite tener desplegado a uno sólo. La forma de conseguir es simplemente añadir a todos los acordeones el atributo name con el mismo nombre.

Mediante CSS, podemos cambiar este comportamiento, haciendo uso del pseudoelemento ::marker sobre la etiqueta summary
