-fronttmater

En ciertos ecosistemas, existe un formato de fichero denominado front-matter. Este formato fue promovido por Jekyll y se basa en una cabecera en formato YAML encerrado en bloques "fenced" mediante ---. El resto del documento es markdown

-Editores o Programas

Por último, existen múltiples editores markdown que se pueden utilizar para escribir documentos en formato md.

Nombre descripcion
Obsidian.md Editor de contenido markdown, con soporte para integraciones y relaciones de elementos.
Zettlr Editor de contenido markdown, ideado para guardar múltiples documentos.
Dillinger Editor de contenido markdown online con soporte para guardar en cloud.
StackEdit Editor de contenido markdown online.
Siyuan Editor de contenido enfocado en la privacidad.
Reor Editor de contenido markdown con soporte para IA local (LLM)
GhostWriter Editor markdown minimalista de KDE.
