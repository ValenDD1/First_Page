Track

track e una etiqueta que sirve para poder mostrar subtitulos de un video,el track e debe almacenar en la etiqueta video(etiqueta mutimedia contenedora) obviamente debemo dar la ruta de nuetro archivo .vtt en el src de nuestro track

ejemplo

hay varias etiquetas que puedes usar

  • src y srclang

    En primer lugar, la etiqueta track debe de incluir un atributo src para indicar donde está nuestro fichero de subtítulos con extensión .vtt. Este atributo es obligatorio. Además, es conveniente establecer también el atributo srclang, mediante el cuál indicaremos el idioma en el que está dicho archivo de subtítulos.

  • label y default

    Si tenemos varias etiquetas track en un mismo archivo de video o audio, podemos darle la opción al usuario para que elija entre múltiples subtítulos. Para ello, el usuario debe ir al botón CC de los controles del video o audio y seleccionar el subtítulo que le interese. Es posible indicar con un atributo label en la etiqueta track el texto que se mostrará en el menú de opciones del botón CC. Observa el siguiente ejemplo con 3 subtítulos diferentes:

  • Kind

    Además, de forma adicional, también podemos indicar el atributo kind para dotar de metadatos o semántica a la etiqueta track. Con este atributo podemos establecer la naturaleza de los subtítulos, indicando si se trata de subtitulos, información, capítulos, etc...
