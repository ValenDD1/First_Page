casillas

es un valor del input que nos permitira que el usuario pueda elegir una o varias opciones mediante una casilla

  • checkbox: Este tipo de control permite mostrar una casilla al usuario, dándole la posibilidad de marcarlo como activado o desactivado.
  • radio: Este tipo de control se suele utilizar cuando el usuario debe elegir sólo una opción específica de varias disponibles.
