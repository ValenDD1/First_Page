Color

color es un valor del atributo tyoe de input y solo sirve para poder ver un color en un pagina web

para usarlo debemos usar el atributo type="color"

para personalizarlo debemos usar los pseudoelementos especiales de css

  • input[type="color"] : Elemento de entrada de datos para elegir color.
  • ::color-swatch-wrapper :Contenedor externo del color elegido.
  • ::color-swatch :Contenedor interno del color elegido.
