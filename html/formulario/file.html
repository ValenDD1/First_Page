File

el atributo type="file" nos permite abrir nuestro fichero de archivos y poder seleccionar,esto nos sirve para poder enviar archivos una base de datos.

para entener algunas cosas del file debemos saber que por defecto se envia "application/x-www-form-urlencoded"

pero hay varias formas de mandar un archivo

  • application/x-www-form-urlencoded
  • multipart/form-data es la mas recomendable
  • text/plain Sólo se envía texto plano. No se suele utilizar hoy en día.
tambien los atributos que usa
  • accept: Indica los tipos de archivos que se pueden subir (image,application,audio,video)
  • accept-charset: Fuerza a utilizar una codificación en los parámetros de texto del formulario.
  • capture: Indica si el usuario puede capturar un archivo desde su dispositivo. (solo en celulares)
  • para form enctype:{"multipart/form-data","application/x-www-form-urlencoded"} ,nos sirve para enviar el archivo y que formato
