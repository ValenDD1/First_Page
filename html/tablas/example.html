tablas

la etiqueta tabla se utiliza para representar una tabla

para poder crear tu primer tabla ,debes saber esas etiquetas
  • tr: Table Row. Etiqueta contenedora de cada fila de la tabla.
  • th: Table Header. Cada una de las celdas de la tabla..
  • td: Table Data. Etiqueta Cada una de las celdas de cabecera de la tabla.
